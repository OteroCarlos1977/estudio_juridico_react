# Estudio Juridico React

Nueva version de la aplicacion Estudio Juridico Rollie, planificada con JavaScript, React, Node.js y SQLite.

## Estructura

- `backend/`: API Node.js y acceso a SQLite.
- `frontend/`: aplicacion React.
- `data/`: base SQLite, adjuntos, reportes y backups locales.
- `docs/`: documentacion del proyecto.
